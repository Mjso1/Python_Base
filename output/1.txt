4️⃣ 프로젝트 폴더 구조 생성
Python_Base/
├── notebooks/      # 주피터 노트북 파일들

└── tests/         # 테스트 파일들

5️⃣ 메인 학습 노트북 생성
01_Python_Basics.ipynb 노트북을 생성하여 8개 섹션으로 구성:

1. 필수 라이브러리 가져오기
   - 시스템 라이브러리 import
   - 데이터 과학 라이브러리 import
   - 환경 정보 및 버전 확인

2. 기본 데이터 타입 다루기
   - 문자열, 숫자, 불린, None 타입
   - 각 타입의 특성과 메서드 활용

3. 변수와 연산자 사용하기
   - 산술, 비교, 논리 연산자
   - 변수 선언과 활용

4. 조건문과 반복문 구현하기
   - if-elif-else 조건문
   - for, while 반복문
   - 리스트 컴프리헨션

5. 함수 정의하고 사용하기
   - 기본 함수 정의
   - 매개변수, 가변인자, 키워드인자
   - 람다 함수와 map 활용

6. 리스트와 딕셔너리 조작하기
   - 리스트 생성, 추가, 제거, 슬라이싱
   - 딕셔너리 조작과 메서드
   - 중첩 데이터 구조 처리

7. 파일 입출력 처리하기
   - 텍스트 파일 읽기/쓰기
   - CSV 데이터 처리
   - 예외 처리를 포함한 안전한 파일 처리

8. 예외 처리 구현하기
   - try-except 기본 문법
   - 여러 예외 타입 처리
   - 사용자 정의 예외 클래스

6️⃣ 유틸리티 모듈 생성
src/utils.py 파일 생성:
- 디렉토리 생성 함수
- JSON/CSV 파일 처리 함수
- 타임스탬프 생성 함수
- 로깅 함수

7️⃣ 테스트 파일 생성
tests/test_utils.py 파일 생성:
- unittest 프레임워크 사용
- 유틸리티 함수들에 대한 단위 테스트

8️⃣ 프로젝트 설정 파일들 생성
- requirements.txt: 패키지 의존성 목록
- README_Project.md: 상세한 프로젝트 가이드 문서
- .gitignore: Git 버전 관리 설정

9️⃣ 대화 내용 보관
- output/conversation_log.txt: 전체 구축 과정 기록
